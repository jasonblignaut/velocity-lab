Lab Prerequisites

Before starting your Velocity Lab journey, ensure you have the following:

  • Create a Microsoft 365 free trial at microsoft.com for Entra ID and Exchange testing (Week 4)
  • An old laptop or desktop with minimum 8GB RAM and 100GB disk space to use as a mock server
  • Install VirtualBox from virtualbox.org, configured with bridged networking (e.g., 192.168.1.x)
  • Request a static IP from your ISP (~R25/month, e.g., Afrihost or Telkom) for external mail access
  • Open firewall ports 25 (SMTP), 587 (SMTPS), 443 (HTTPS), and 993 (IMAPS) on virtual servers, laptop, and router
  • Ensure stable internet connection with minimum 10 Mbps for Microsoft 365 integration and updates
