Your 4-Week Learning Path

Week 1: Foundation Setup

Build your Windows Server 2012 domain controller, join virtual machines, and create secure network shares with automatic drive mapping.

Week 2: Infrastructure Expansion

Deploy your second domain controller for redundancy, implement WSUS for centralized updates, and configure enterprise time synchronization.

Week 3: Exchange Deployment

Upgrade to Server 2016, install Exchange Server 2019, create user mailboxes, and establish reliable internal mail flow between users.

Week 4: Hybrid Migration

Configure external mail publishing, implement SSL certificates, and complete your Microsoft 365 hybrid setup with successful Exchange Online migration.

  • DNS management access for your test domain
