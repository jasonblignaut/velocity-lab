Lab Prerequisites

  • Create a Microsoft 365 free trial at microsoft.com for Entra ID and Exchange testing (Week 4).
  • An old laptop (minimum 8GB RAM, 100GB disk) to use as a mock server.
  • Install VirtualBox from virtualbox.org, configured with bridged networking (e.g., 192.168.1.x).
  • Request a static IP from your ISP (~R25/month, e.g., Afrihost or Telkom) for external mail access.
  • Open firewall ports 25 (SMTP), 587 (SMTPS), 443 (HTTPS), and 993 (IMAPS) on virtual servers, laptop, and router (port forward to laptopâ€™s IP).
  • Ensure stable internet (minimum 10 Mbps) for Microsoft 365 and updates.
